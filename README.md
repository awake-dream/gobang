# Gobang 五子棋游戏

一个基于浏览器的五子棋游戏，使用原生 JavaScript 开发。

## 功能

- 人机对战模式
- 棋盘绘制与交互
- 胜负判定

## 运行方式

```bash
npm install
npm run dev
```

## 项目结构

```
gobang/
├── index.html      # 主页面
├── src/
│   ├── board.js     # 棋盘绘制
│   ├── game.js      # 游戏逻辑
│   ├── main.js      # 入口文件
│   └── style.css    # 样式文件
├── jsconfig.json
└── package.json
```
